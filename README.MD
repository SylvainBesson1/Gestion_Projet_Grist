# Widget Kanban pour Grist

Un widget Kanban configurable pour Grist, permettant de gérer des tâches avec des colonnes dynamiques.

## Installation

1. Copiez l'URL brute de ce fichier : [index.html](https://raw.githubusercontent.com/SylvainBesson1/Gestion_Projet_Grist/refs/heads/main/index.html)
2. Dans Grist, allez dans **Add New > Widget from URL**.
3. Collez l'URL et validez.

## Configuration

Dans le panneau droit du widget, sélectionnez :
- La table des tâches.
- Les colonnes pour le titre, la description, la priorité, l'état, etc.

## Fonctionnalités

- Glisser-déposer des tâches entre les colonnes.
- Filtres par priorité, état, projet, et dossier.
- Création/édition/suppression de tâches.
- Affichage des deadlines et des assignations.
